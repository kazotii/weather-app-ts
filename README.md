# Weather App (TypeScript)
A weather forecasting application that provides real-time weather data.

## Tech Stack
* Core: React, TypeScript, Vite
* State Management: Custom Hooks / Store logic (useWeatherStore)
* API: Integration with weather data providers
* Styling: CSS

## Features
* Location Search: Search for weather information by city name.
* Real-time Data: Displays current weather conditions and temperature.
* Visual Representation: Weather cards for clear data visualization.
* Type Safety: Built with TypeScript for reliable data handling.
